# Contributing

Thanks for your interest in embraOS-QNM-Core!

## Bug Reports & Feature Requests

We welcome bug reports and feature requests via [GitHub Issues](https://github.com/Ward-Software-Defined-Systems/embraOS-QNM-Core/issues). Please include:

- A clear description of the problem or suggestion
- Steps to reproduce (for bugs)
- Your environment (OS, hardware/accelerator, and relevant tool/runtime versions)

## Pull Requests

**We are not accepting external pull requests at this time.** This project is actively developed by Ward Software Defined Systems and follows an internal development workflow.

You're welcome to:

- **Open issues** to suggest changes — if we agree, we'll implement them
- **Star** the repo if you find it useful

## Code of Conduct

Be respectful and constructive in all interactions. We reserve the right to close issues or block users who are disruptive.
